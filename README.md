This is a collection of some of the Mini Challenges inside of Scrimba's Full Stack Development course.
Throughout the course, you learn each module through a project. However, to learn new functions or ideas, it usually has you do 1 quick challenge with the function or idea first, then implement it in your project.
This repo is a collection of some of the quick challenges. I only add the ones that are the most important since there are so many quick challenges.
